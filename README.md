# rebel-puzzles

Interactive puzzles for [rebel-the-dog.com](https://rebel-the-dog.com), hosted via GitHub Pages at [twescott.github.io/rebel-puzzles](https://twescott.github.io/rebel-puzzles).

## Structure

- `index.html` — Puzzle index page
- `puzzles/` — One HTML file per puzzle, self-contained using [Exolve](https://github.com/viresh-ratnakar/exolve)

## Adding a Puzzle

Each puzzle is a standalone HTML file in `puzzles/`. The agent handles this automatically from a `.puz` file.
